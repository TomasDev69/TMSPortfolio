Certify

Premium, verifiable course certificates regenerated from any Skilljar credential. A user pastes an official validation URL; Certify verifies it in real time, then regenerates a print-ready, dynamically themed PDF with a custom QR code, no database and zero forgery surface. Built end-to-end with Claude Code. Stack: Next.js (App Router), React, TypeScript, Tailwind CSS, Cheerio, html2canvas, jsPDF, Vercel.

  • Credential validation in a server-side `/api/verify` route that is SSRF-safe, using Cheerio to scrape the live Skilljar DOM and bypass CORS to extract trusted metadata.

Raspberry Pi 5 Home Custom Server (NAS)

RPI 5 Home Custom Server (NAS)

The project consists of building a custom Network Attached Storage (NAS) connected to the home router via Ethernet, based on Raspberry Pi 5 and configured with OpenMediaVault, housed in a 4-bay 3D-printed case. Each bay hosts a 4 TB disk; RAID 10 configuration is being evaluated for mirroring, reliability, and good performance, with a capacity efficiency of 50% due to the mirroring of purchased disks in pairs. The current bottleneck is the network: the PC operates via Wi-Fi with an effective speed of around 100 Mbit/s (~12 MB/s), while a direct Ethernet connection would allow reaching up to ~800 Mbit/s. Considering the expected usage (file sharing with friends and collaborators), the current performance remains adequate during the initial phase.

Il progetto consiste nella realizzazione di un Network Attached Storage (NAS) custom collegato al router domestico tramite Ethernet, basato su Raspberry Pi 5 e configurato con OpenMediaVault, alloggiato in un case stampato in 3D a 4 baie. Ogni baia ospita un disco da 4 TB; è in valutazione una configurazione RAID 10, scelta per garantire mirroring, affidabilità e buone prestazioni, con un’efficienza di capacità del 50% dovuta al mirroring dei dischi acquistati a coppie. Attualmente il principale collo di bottiglia è la rete: il PC opera via Wi-Fi con una velocità effettiva di circa 100 Mbit/s (~12 MB/s), mentre un collegamento Ethernet diretto permetterebbe di raggiungere fino a ~800 Mbit/s. Considerato l’utilizzo previsto (condivisione di file con amici e collaboratori) le prestazioni attuali risultano comunque adeguate nella fase iniziale.
