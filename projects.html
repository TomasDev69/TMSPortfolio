Certify

Premium, verifiable course certificates regenerated from any Skilljar credential. A user pastes an official validation URL; Certify verifies it in real time, then regenerates a print-ready, dynamically themed PDF with a custom QR code, no database and zero forgery surface. Built end-to-end with Claude Code. Stack: Next.js (App Router), React, TypeScript, Tailwind CSS, Cheerio, html2canvas, jsPDF, Vercel.

Certificati di corso premium e verificabili, rigenerati a partire da qualsiasi credenziale Skilljar. L'utente incolla un URL di validazione ufficiale; Certify lo verifica in tempo reale e rigenera un PDF con tema dinamico, pronto per la stampa, con QR code personalizzato, senza database e senza superficie di contraffazione. Sviluppato interamente con Claude Code. Stack: Next.js (App Router), React, TypeScript, Tailwind CSS, Cheerio, html2canvas, jsPDF, Vercel.
