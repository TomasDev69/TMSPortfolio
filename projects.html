Certify

Premium, verifiable course certificates regenerated from any Skilljar credential. A user pastes an official validation URL; Certify verifies it in real time, then regenerates a print-ready, dynamically themed PDF with a custom QR code, no database and zero forgery surface. Built end-to-end with Claude Code. Stack: Next.js (App Router), React, TypeScript, Tailwind CSS, Cheerio, html2canvas, jsPDF, Vercel.

Technical Highlights:

  • Credential validation in a server-side `/api/verify` route that is SSRF-safe, using Cheerio to scrape the live Skilljar DOM and bypass CORS to extract trusted metadata.
  • SSRF-safe credential validation in a server-side /api/verify route, scraping the live Skilljar DOM with Cheerio to bypass CORS and extract trusted metadata.
  • Defeated an html2canvas kerning conflict with Next.js-optimized fonts by injecting CDN fonts into the render clone before rasterization, for high-fidelity PDF export.
  • Dynamic per-course theming for 18+ courses via CSS variables, plus local on-the-fly QR-code generation.
  • Dynamic per-course theming for 18+ courses via CSS variables, plus local on-the-fly QR-code generation.
  • Fully server-rendered and stateless on Vercel, with a strict dark-minimalist, high-contrast design system.

Local AI Server Hosting

This project concerns the local hosting of artificial intelligence (AI) services on an entirely managed infrastructure at home. The objective is to create a secure and performant environment for running AI applications, avoiding dependence on external cloud services. Using dedicated hardware and optimized configurations, the project aims to provide a fast and reliable AI experience, while also ensuring data privacy and full control over processes. This setup is ideal for independently testing, developing, and implementing AI models securely.

Questo progetto riguarda l'hosting locale di servizi di intelligenza artificiale (AI) su un'infrastruttura gestita interamente in casa. L'obiettivo è creare un ambiente sicuro e performante per eseguire applicazioni AI, evitando la dipendenza da servizi cloud esterni. Utilizzando hardware dedicato e configurazioni ottimizzate, il progetto mira a fornire un'esperienza AI rapida e affidabile, garantendo al contempo la privacy dei dati e il controllo completo sui processi. Questo setup è ideale per testare, sviluppare e implementare modelli AI in modo autonomo e sicuro.

Conoscenze acquisite:

  • Configuration and management of local AI servers
  • High-performance hardware management (GPU)
  • AI applications setup and testing
  • Backup and recovery of data and AI models
